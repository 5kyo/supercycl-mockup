# Supercycl Mockup — App Source

앱 소스 코드 디렉토리. 프로젝트 소개는 [루트 README](../README.md)를 참고하세요.

## Commands

```bash
npm install       # 의존성 설치
npm run dev       # 개발 서버
npm run build     # 프로덕션 빌드
npm run lint      # 린트
npm run preview   # 빌드 미리보기
```
