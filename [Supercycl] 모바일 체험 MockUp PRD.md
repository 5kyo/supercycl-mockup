> **문서 버전**: v2.0
> **작성일**: 2026.03.31
> **작성자**: 권오교
> **대상 행사**: 2026.04.11 유스메타 협업 행사
> **목적**: 이해관계자 논의용 모바일 MockUp 화면 정의

## 1. 개요

### 1.1 배경

4/11 유스메타 협업 행사에서 1,000명+ 선물 거래 이용자가 모바일로 Supercycl을 체험합니다. 현재 모바일 버전이 없으므로 **이해관계자 논의용 모바일 목업 웹**을 제작하여 화면 구성과 UX 플로우를 사전 합의합니다.

> **본 문서의 범위는 "목업"입니다.** 실제 백엔드 연동, API 호출, 지갑 생성, 자금 전송 등은 포함하지 않습니다. 모든 데이터는 하드코딩된 더미 데이터이며, 화면 전환과 UI 인터랙션만 동작합니다. 실제 동작하는 체험 빌드는 목업 합의 후 별도 PRD로 진행합니다.

### 1.2 행사 운영 구조

행사는 **두 트랙**으로 병렬 운영됩니다.

| 트랙 | 환경 | 대상 | 내용 |
|------|------|------|------|
| PC 시연 | 상용 Supercycl 웹 | 노트북 지참 유저 | CEX 다중 연동, BingX/Binance/OKX API 연동 실시간 시연 |
| 모바일 체험 | **임시 모바일 웹 (본 문서 범위)** | 전체 참석자 1,000명+ | QR → 가입 → 거래까지 2분 이내 체험 |

> 본 PRD는 **모바일 체험 트랙의 목업**을 다룹니다. PC 시연은 상용 버전을 그대로 사용합니다.

### 1.3 목업의 목적과 범위

| 구분 | 목업 (본 문서) | 체험 빌드 (추후) |
|------|---------------|-----------------|
| **목적** | 이해관계자 간 화면 구성/UX 플로우 합의 | 행사 당일 실제 유저 체험 |
| **데이터** | 하드코딩 더미 데이터 | 실제 API 연동 |
| **로그인** | 시뮬레이션 (버튼 클릭 → 다음 화면 이동) | 실제 Google OAuth |
| **지갑/자금** | 로딩 애니메이션만 (실제 생성 X) | 백엔드 자동 생성 + 자금 전송 |
| **거래** | UI 인터랙션만 (체결 X) | HL Testnet 실제 체결 |
| **배포** | 로컬 또는 정적 호스팅 | 체험 전용 도메인 |

체험 빌드의 목표 소요시간은 QR 스캔 → 가입 → 첫 거래까지 **2분 이내**이며, 목업에서는 이 플로우의 **화면 단위 전환 흐름**을 검증합니다.

### 1.4 핵심 전제

| 항목 | 내용 |
|------|------|
| 플랫폼 | 모바일 웹 (iOS Safari, Android Chrome 대응) |
| 성격 | **이해관계자 논의용 인터랙티브 목업**. 백엔드 연동 없음. |
| 데이터 | 모든 가격, 잔고, 포지션은 **하드코딩 더미 데이터** |
| 로그인 | 시뮬레이션 (버튼 탭 → 화면 전환만) |
| 거래 | UI 인터랙션만 동작 (실제 주문 체결 없음) |
| 파트너 기능 | 2배 레버리지 고정 + 자동 TP/SL **UI 시연** |
| 기술 스택 | 프론트엔드 단독 (React/Next.js 또는 정적 HTML) |

### 1.5 대상 유저

BingX, Binance, OKX 등 CEX 선물 거래 경험자. DEX/지갑에 익숙하지 않을 수 있으며, 모바일에서 최소한의 터치로 가입~거래까지 완료해야 합니다.

## 2. 유스메타 측 요구사항 (목업에서 시연할 항목)

두찬실장님과의 협의를 통해 확인된 요구사항입니다. 목업에서는 아래 항목의 **UI와 화면 전환 흐름**을 시연합니다.

| # | 구분 | 요구사항 | 목업에서의 시연 방식 |
|---|------|----------|---------------------|
| 1 | **필수** | QR 기반 현장 가입 체험 | 랜딩 → 로그인 → 온보딩 화면 전환 플로우 시연 (실제 OAuth 없음) |
| 2 | **필수** | 레버리지 2배 고정 | 안내 팝업 UI + Adjust Leverage 모달에서 2배 제한 슬라이더 시연 |
| 3 | **필수** | 자동 TP/SL 체험 | 설정 모달 → Order Form 인디케이터 → Dashboard "Auto" 배지 UI 시연 |
| 4 | **필수** | 레퍼럴 코드 자동 등록 | URL에 ref=YOUTHMETA 파라미터 포함된 상태로 랜딩 화면 시연 |
| 5 | **희망** | 사전 테스트 환경 (D-3) | 목업 URL 공유하여 유스메타 측 사전 리뷰 (최문창 상무, 두찬실장님) |
| 6 | **참고** | PC 시연 (CEX 연동) | 본 목업 범위 밖. 상용 PC 버전에서 별도 진행. |

> **유스메타 측 확인 필요**: Auto TP/SL 기본 ON/OFF 상태, 안내 팝업 언어(영문/국문 병기), 목업 리뷰 가능일
